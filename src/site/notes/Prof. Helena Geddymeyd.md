# Charakterbeschreibung
Prof. Helena Geddymeyd ist eine Koryphäe auf dem Feld der Beschwörungsmagie und verhält sich auch wie eine. Sie lehrt an der Platinum Akademie und hält dort zentrale Vorlesungen zur Beschwörung von Konstrukten und den unterschiedlichen Planes. Sie ist oft ungeduldig, unnachgiebig und kann sehr gereizt, werden, wenn die von ihr vorgegeben Ziele nicht erreicht werden. Sie hat hohe Ambitionen innerhalb der Akademie und möchte auch hier ihren Einfluss ausbauen.

# Beziehung, Vorgeschichte zu Adda
Adda betrachtete sie zu Anfang nicht besonders, wohl aber, nachdem ihr besondere Leistungen nachgesagt wurden. Auch wenn Adda um einiges jünger ist, und Helena bereits seit Jahren eine feste Größe in der Akademie darstellt, so sieht sie Addas aufstreben kritisch. Wenn es nach ihr ginge, dann sollten alle Schüler eine feste Anzahl an Jahren an der Akademie gelernt haben, bevor sie auch nur annähernd Aussichten auf eine Laufbahn als Professor an der Akademie bekämen. Von Ausnahmen von diesem Prozess, wie sie Adda eine bekommen hat, hält sie wenig und gibt das auch auf zu verkunden.
