# Charakterbeschreibung
Eine Sergeant der Stadtwache. Sie ist für ihr kaltes, professionelles und kühles Auftreten bekannt. Meistens ist sie mit einer Truppe Wachen zu sehen. Sie ist eine Vorgesetzte von Kathkalo.
